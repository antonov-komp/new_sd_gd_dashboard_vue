Тест маршрутов секторов

Сектор 1С Сектор PDM Сектор Bitrix24 Сектор Infrastructure Главная

Нажмите на ссылки выше, чтобы протестировать маршруты секторов.

Ожидаемый результат:

  • Сектор 1С: Должен показать реальные данные из Bitrix24
  • Сектор PDM: Должен показать структуру с этапами: Проектирование, Проверка, Внедрение
  • Сектор Bitrix24: Должен показать структуру с этапами: Анализ, Разработка, Тестирование, Внедрение
  • Сектор Infrastructure: Должен показать структуру с этапами: Заявка, Оценка, Закупка, Внедрение

Отладочная информация:

Проверьте консоль браузера (F12) для логов загрузки данных секторов.

## Чек-лист при переезде на новый сервер

Используйте [SERVER-MIGRATION-GUIDE.md](./SERVER-MIGRATION-GUIDE.md) для полного чек-листа.

**Основные шаги:**

1. ✅ Изменить `base: './'` в `vite.config.js`
2. ✅ Добавить тег `<base>` в `index.php`
3. ✅ Обновить все сервисы для использования `path-utils.js`
4. ✅ Исправить глобальные стили в `main.css`
5. ✅ Пересобрать приложение: `npm run build`
6. ✅ Проверить работу всех модулей

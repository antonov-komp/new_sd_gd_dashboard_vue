## Описание

Обновить fallback-сообщение для пустого состояния нулевой точки. Если на уровне стадии нет тикетов по сотруднику 1051 (Хранитель объектов), должно отображаться сообщение: "Нет неразобранных тикетов в стадии".

**Текущее отображение:**
```vue
<div v-if="tickets.length === 0" class="empty-state">
  <p>Нет входящих тикетов</p>
</div>
```

**Новое отображение:**
```vue
<div v-if="tickets.length === 0" class="empty-state">
  <p>Нет неразобранных тикетов в стадии</p>
</div>
```

**Проверка отображения тикетов:**
- Тикеты без ответственного отображаются в нулевой точке

### 3. Интеграционное тестирование

- Маппинг полей `assignedById` и `ASSIGNED_BY_ID` работает корректно
- Кеширование данных работает корректно (если используется)

### 4. Тестирование Drag & Drop

**Проверка перемещения тикетов:**

- Перетаскивание тикета без ответственного из нулевой точки на сотрудника работает
- Перетаскивание тикета между сотрудниками работает
- Перетаскивание тикета между этапами работает

### 5. Регрессионное тестирование

- Все три этапа отображаются корректно

- Обработка ошибок работает корректно
